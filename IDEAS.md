# Ideas

These are feature ideas and observations gathered during development.

Inclusion here does not imply priority or commitment.

---

## Search

### Predictive Search

Investigate predictive / incremental searching similar to the Creation Kit Object Window.

Possible behaviours:

- live filtering
- auto-complete
- ranked suggestions
- recent searches

---

### Multiple Search Tabs

Allow multiple search tabs.

Possible use cases:

- Reactor sounds
- UI sounds
- Creature sounds

Similar to:

- CK Object Window
- Foobar2000 playlists
- Browser tabs

---

## Playback

### Loop Modes

Support:

- no loop
- loop current file
- loop current queue

---

### Queue Playback

Play sounds sequentially from:

- search results
- collections
- favourites
- history

---

### Sorting

Allow queue ordering by:

- filename
- path
- random
- recently added
- custom order

---

### Play Entire Library

Allow playback across the current filtered result set.

Examples:

- entire indexed library
- current search
- current collection
- current tag

Playback follows the current sort order.
